Our Work

Currently there is no system that can verify the content of Holy Quran accurately.The system that currently exists verifies ayats without rasmul-khat and use a technique of exact analysis.
In this system,there are three ways a user can verify an ayah or a part of ayah using:

    Audio

  • The user have to speak or upload an audio file which will be recognized via python library(Google Speech Recognition) to verify the content.

    Image

  • The user have to upload an image file which will be recognized via python library (TesseractOCR) to verify the content.

    Text

  • The user will have to paste an Arabic text(with or without tashkeel) or type(without taskeel) through a keyboard attached to the search bar.

    Wrong Input Correction

  • Also if the input is wrong,it will be corrected via python library (Fuzzy Wuzzy).

    Algorithms for Searching and display

  • A String Matching algorithm is used for searching the input in database and A priority Based algorithm is being used in this system to display the result for best matches.
